# 🪵 Decorê Arts

> Artes em madeira personalizadas e exclusivas, unindo o rústico ao sofisticado diretamente na sua decoração.

O **Decorê Arts** é uma plataforma web (e-commerce/catálogo digital) desenvolvida para apresentar, divulgar e comercializar peças artesanais exclusivas feitas em madeira. O projeto foi desenhado com foco na experiência do utilizador, proporcionando uma navegação fluida, visualização detalhada dos produtos e um processo simples de encomenda ou contacto.

🌐 **Acesse a aplicação:** [decore-arts.vercel.app](https://decore-arts.vercel.app/)

---

## 🚀 Funcionalidades Principais

* **Catálogo Digital Interativo:** Visualização de peças de artesanato organizadas por categorias, com imagens de alta qualidade e descrições detalhadas.
* **Personalização sob Encomenda:** Secção ou formulário dedicado para utilizadores solicitarem peças exclusivas com medidas ou gravações personalizadas.
* **Design Responsivo & Elegante:** Interface totalmente adaptada para dispositivos móveis, tablets e computadores, utilizando uma palete de cores quente e minimalista que reflete a essência do trabalho com madeira.
* **Integração de Contactos:** Atalhos diretos para canais de atendimento (WhatsApp, Redes Sociais ou E-mail) para fecho de vendas e orçamentos.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **Frontend:** [React.js](https://reactjs.org/) / [Next.js](https://nextjs.org/) (ou a tecnologia que usou, ex: HTML5/CSS3/JavaScript)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/) / Styled Components
* **Alojamento & Deploy:** [Vercel](https://vercel.com)
* **Ícones:** [Lucide React](https://lucide.dev/) / React Icons
